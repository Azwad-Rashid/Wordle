# WORDLE
Implementation of the popular word game "Wordle" using the Tkinter module

### Dependencies:
1. Tkinter v8.6.13

### How to play:
1. Run the main.py file using `py main.py`
2. And that's it!

### Documentation:
Open `docs/_build/index.html` to see the documentation of the codebase
